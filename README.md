# DevOps Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

## About

Hi, I'm **Shiva Rajan** — a DevOps Engineer passionate about building resilient infrastructure, automating deployments, and bridging the gap between development and production.

## Pages

- **Home** — About me and areas of expertise
- **Resume** — Work experience, education, skills, and certifications
- **Portfolio** — Projects and certifications with detailed case studies
- **Contact** — Get in touch

## Tech Stack

- HTML5 / Vanilla CSS / JavaScript
- Tailwind CSS (CDN)
- Google Fonts — Space Grotesk & Inter
- Material Symbols Outlined

## Features

- Dark / Light mode toggle (persists across pages)
- Fully responsive — desktop, tablet, and mobile
- Interactive project overlay modals
- Portfolio filter by type (Project / Certification / Badge)

## How to Run

Open `index.html` in any modern browser. No build step required.
